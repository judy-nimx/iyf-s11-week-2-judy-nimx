## Project Description
A clean, semantic basic landing page built to establish my online presence as a student developer. This project serves as the foundation for my multi-page personal portfolio website, designed to showcase my journey, skills, and future development projects.

## Technologies Used
- HTML5

## Features
- **Semantic Structure:** Uses modern HTML5 tags like `<header>`, `<nav>`, `<main>`, `<section>`, and `<footer>`.
- **Navigation Menu:** Built-in links to connect Home, About, Projects, and Contact pages.
- **Hero Section:** A clean introductory display highlighting my background and goals.

## How to Run
1. Clone this repository to your local machine.
2. Open `index.html` directly in your web browser.

## Lessons Learned
I learned how to structure an HTML document logically using semantic elements. I also practiced setting up clean navigation links that will make it easy to expand the website into a multi-page platform later on.

## Challenges Faced
- **Challenge:** Making sure the file paths for the navigation links (`about.html`, `projects.html`, etc.) were relative so they can load correctly across different folders.
- **Solution:** I carefully mapped out the planned folder layout and used standard relative URLs.
